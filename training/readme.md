# Setup for training

## Traffic Signs Detection
**1. Move dataset '_trafficSigns_' to '_datasets_' folder in this project**
**2. Modify config file for Ultralytics**
   * Search '_%appdata%_' file folder in File Explorer and open.<br>
   * Search '_Ultralytics_' folder and open.<br>
   * Open '_settings.yaml_' file and set '_dataset-dir_' key with your path for this project. <br>
   * Your path should look like this: '_C:\Users\...\CarVision\training\datasets_'.
**3. Run *train.py* & wait.**

Results will be saved to '_results\\traffic-signs-detection_' folder.
