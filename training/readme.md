# Setup for training

## Traffic Signs Detection
1. move dataset 'trafficSigns' to 'datasets' folder in this project
2. search '%appdata%' file folder in File Explorer and open
3. search 'Ultralytics' folder and open
4. open 'settings.yaml' file and set 'dataset-dir' key with your path for this project as value; <br>
   your path should look like this: 'C:\Users\...\CarVision\training\datasets'
5. run 'train.py' & wait 

Results will be saved to 'results\\traffic-signs-detection' folder.
